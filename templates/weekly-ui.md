Audit the Velum docs UI for drift.

Check the following:

1. Spacing
- Are sections consistently spaced?
- Has vertical rhythm compressed?

2. Typography
- Are heading sizes consistent?
- Is body text readable and consistent?

3. Image treatment
- Are images still primary content?
- Have borders, shadows, or cards been introduced?

4. Layout density
- Has content become more crowded?
- Are sections still breathing?

5. Motion
- Is animation still subtle?
- Has motion become distracting?

6. Design system adherence
- Are tokens being used consistently?
- Are there arbitrary values creeping in?

Output:
- list of issues
- suggested fixes
- severity (low/medium/high)

Goal:
Prevent gradual degradation of the editorial experience.
